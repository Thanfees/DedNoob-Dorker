Add keywords for each operator to build advanced queries. Leave any field blank to skip that operator.

Mix operators to narrow results. Example: `site:example.com AND inurl:login AND allintext:"reset password"`.

Google Dork Operators Cheat Sheet

allintext
Finds pages where all specified keywords appear in the body text.
intext
Locates pages that include any of the keywords within the body content.
inurl
Searches for URLs containing at least one of the provided keywords.
allinurl
Returns URLs that include every keyword listed in the query.
intitle
Targets pages whose titles contain any of the given keywords.
allintitle
Matches pages with titles that include all keywords together.
link
Finds pages that link externally to the specified URL or keyword.
